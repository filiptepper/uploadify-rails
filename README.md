# Uploadify::Rails

Uploadify plugin for Ruby on Rails asset pipeline.

## Installation

Add this line to your application's Gemfile:

    gem 'uploadify-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install uploadify-rails

## Usage

Add to your JavaScript pipeline:

    //= require uploadify

Add to you CSS pipeline:

    /*
     *= require uploadify
    */
